# Otonom Video Dönüştürücü

Google Colab üzerinde GPU (Ekran Kartı) gücünü kullanarak videoları yüksek kaliteli HEVC 10-Bit formatına dönüştürür.

## Özellikler
- **Görüntü:** H.265 (HEVC) 10-Bit (Renk halkalanmasını önler).
- **Ses:** libopus (Yüksek verimli ses sıkıştırma).
- **Kontrol:** İşlemi duraklatma ve devam ettirme desteği.

## Nasıl Kullanılır?
1. Google Drive'ınızı bağlayın.
2. `pip install -r requirements.txt` komutuyla gerekli araçları yükleyin.
3. Kodu çalıştırıp klasör yolunuzu girin.
